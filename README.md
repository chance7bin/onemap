# onemap

## 项目介绍

项目选择OpenLayers为前端地图库、GeoServer为地图服务器、PostgreSQL与PostGIS为数据库的开源GIS体系结构，基于uni-app跨平台开发框架设计并实现交通移动一张图原型系统。
